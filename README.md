# 轻签

轻签是一款完全纯本地的 Android 习惯签到 App。

## 功能

- 创建、编辑、暂停、恢复和删除习惯
- 每天或每周指定日期重复
- 今日签到和撤销
- 今日进度与每日总结
- 连续签到、累计次数、本周完成率、最近 7 天和月历
- 后台与锁屏提醒
- 跟随系统、浅色和深色主题
- 本地数据一键清空

## 下载

当前版本：`v1.0.0`

最低系统：Android 13

## 安装与提醒权限

首次安装后，请在轻签“设置”中：

1. 开启通知权限。
2. 开启准时提醒权限。
3. 将轻签设为不允许电池优化。
4. 荣耀手机需关闭轻签的自动启动管理，并允许自启动、关联启动和后台活动。

详细步骤见 [安装说明](releases/v1.0.0/安装说明.md)。

## 隐私

- 不需要登录。
- 不包含联网权限。
- 习惯、签到记录和提醒设置仅保存在当前手机本地。
- 不会上传数据到服务器。

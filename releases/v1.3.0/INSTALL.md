# 轻签 1.3.0 安装说明

## 系统要求

- Android 13 或更高版本
- 安装时允许浏览器、文件管理器或 GitHub 下载的 APK 安装未知来源应用

## 版本说明

这是 `1.3.0` 正式版，主要新增：

- 新增独立“记录”页面
- 记录项目支持每天、每月或指定日期列表频次
- 指定日期可通过系统日历逐条选择
- 记录项目支持数字记录、单位、提醒开关和提醒时间配置
- 每个记录项目默认显示最近 15 天折线图
- 记录模块不和习惯签到绑定

## 从旧版本升级

1. 下载并打开 `qingqian-v1.3.0.apk`。
2. 选择覆盖安装。
3. 原有习惯、签到记录、提醒和设置会保留。

## 首次安装

1. 打开 `qingqian-v1.3.0.apk` 并按系统提示完成安装。
2. 在轻签“设置”中允许通知权限。
3. 开启准时提醒权限。
4. 将轻签设为不允许电池优化。
5. 荣耀手机还需要关闭轻签的自动启动管理，并允许自启动、关联启动和后台活动。

## 数据与隐私

- 轻签不需要登录，也不包含联网权限。
- 所有习惯、签到、提醒和记录数据仅保存在当前手机本地。
- 卸载 App 或在设置页清空数据后，数据无法恢复。

# 轻签 1.0.0 安装说明

## 系统要求

- Android 13 或更高版本
- 安装时允许浏览器或文件管理器安装未知来源应用

## 首次安装

1. 在手机上打开 `轻签-v1.0.0.apk`。
2. 按系统提示完成安装。
3. 打开轻签，在“设置”中允许通知权限。
4. 开启“准时提醒权限”。
5. 将轻签设为“不允许电池优化”。
6. 荣耀手机还需进入“应用启动管理”，关闭轻签的自动管理，并允许自启动、关联启动和后台活动。

## 从开发测试版切换

如果手机已经安装开发测试版，正式版会因为签名不同而无法直接覆盖。

切换步骤：

1. 卸载手机上的开发测试版轻签。
2. 安装 `轻签-v1.0.0.apk`。
3. 重新完成通知和后台运行权限设置。

卸载开发测试版会删除其中的习惯、签到和提醒数据。

## 数据与隐私

- 轻签不需要登录。
- 轻签不包含联网权限。
- 所有习惯、签到和提醒设置仅保存在当前手机本地。
- 卸载 App 或在设置页清空数据后，数据无法恢复。

## 后续升级

后续正式版本可以直接覆盖安装，但必须使用轻签的同一正式签名密钥，并使用更高的版本号。

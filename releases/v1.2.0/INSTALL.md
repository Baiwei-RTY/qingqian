# 轻签 1.2.0 安装说明

## 系统要求

- Android 13 或更高版本
- 安装时允许浏览器、文件管理器或 GitHub 下载的 APK 安装未知来源应用

## 版本说明

这是 `1.2.0` 正式版，主要新增：

- 每月指定日期签到
- 习惯页模糊搜索
- 桌面多习惯签到小组件，可上下滑切换进行中的习惯并直接签到
- 小组件使用习惯主题色背景

## 从旧版本升级

1. 下载并打开 `qingqian-v1.2.0.apk`。
2. 选择覆盖安装。
3. 原有习惯、签到记录、提醒和设置会保留。

## 首次安装

1. 打开 `qingqian-v1.2.0.apk` 并按系统提示完成安装。
2. 在轻签“设置”中允许通知权限。
3. 开启准时提醒权限。
4. 将轻签设为不允许电池优化。
5. 荣耀手机还需要关闭轻签的自动启动管理，并允许自启动、关联启动和后台活动。

## 数据与隐私

- 轻签不需要登录，也不包含联网权限。
- 所有习惯、签到和提醒设置仅保存在当前手机本地。
- 卸载 App 或在设置页清空数据后，数据无法恢复。
